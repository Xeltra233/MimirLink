  - send `stream: false`
  - send `output_format: 'hex'`
  - coerce `pitch` with `Math.round(...)`
  - decode returned audio using `Buffer.from(audio, 'hex')`
  - surface `base_resp.status_msg` when MiniMax returns a non-zero status code
